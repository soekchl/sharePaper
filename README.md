# sharePaper

https://share.utcgame.com/

#### 介绍
共享写字板 使用websock共享每个人写的内容

##### v0.2.1
	增加 - 编辑文本内容 对其他用户加锁

#### v0.2.0
	多个编辑框文本传输

#### v0.1.0
	单个编辑框文本传输


#### 安装教程

1. git clone https://gitee.com/soekchl/sharePaper.git
2. cd sharePaper
3. go build
4. ./sharePaper :8080
